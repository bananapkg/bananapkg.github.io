O que é?

BananaPKG é um gerenciador de pacotes de baixo nível escrito em Shell bash que é executado via CLI, com ela é possível Criar, Instalar, Remover, Atualizar e Procurar pacotes no sistema.
Foi criado especialmente para rodar na distribuição Brasileira from scratch mazonOS, porém é possível executar em qualquer distribuição Linux, desde que as dependências sejam supridas.
O seu formato de pacote é .mz, um simples tarball comprimido com a ferramenta xz. Para mais detalhes consulte a documentação.

KISS + UNIX WAY

O banana segue a filosia KISS e UNIX WAY, o programa é extremamente direto e simples, faz somente o que foi criado para fazer! gerenciar pacotes. Nada menos nada mais, a complexidade é descartada por natureza.

Roda em qualquer lugar

Se atender os requisitos que estão abaixo o banana roda em qualquer distribuição Linux. Foi testado em algumas distribuições:
  • mazonOS [Padrão]
  • LFS
  • Slackware
  • Debian
  • Arch Linux

Seguro

A segurança é um bem que o banana não deixa de lado, muito bem testado e verificado cortando as principais artimanhas que os hackerzinhos de quintal usam.

Dependência

  • Bash >= 4.4.18
  • GNU Sed >= 4.2.2
  • GNU Tar >= 1.30
  • xz >= 5.2.2
  • GPG >= 1.4.23
